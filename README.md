# website

Marketing website for DubCall — a static React + Vite app.

## Pages

- `/` — Home
- `/company` — Company
- `/product` — Product
- `/pricing` — Pricing
- `/contact` — Contact
- `/privacy` — Privacy
- `/resources` — Resources

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
